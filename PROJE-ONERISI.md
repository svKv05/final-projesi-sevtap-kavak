Proje Önerisi

Görev Numarası: 3

Ürün Adı:
Öğrenci Başarı Risk Analizi Sistemi

Problem:
Öğrencilerin başarısız olma riskini önceden tahmin etmek.

Hedef Kullanıcı:
Öğrenciler ve eğitim kurumları.

Veri Kaynağı:
UCI Machine Learning Repository - Student Performance Dataset.

Teknolojiler:
Python
Pandas
Scikit-learn
SHAP
Streamlit

Beklenen Çıktı:
Kullanıcı öğrenci bilgilerini sisteme girer. Sistem öğrencinin başarı riskini tahmin eder ve bu tahminin oluşmasında etkili olan faktörleri kullanıcıya açıklar.

Farklılaştırıcı Yön:
Sistem yalnızca tahmin üretmekle kalmayacak, SHAP yöntemi kullanılarak tahmin sonucunun hangi değişkenlerden etkilendiği de açıklanacaktır.
